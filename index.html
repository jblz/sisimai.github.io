What is Sisimai ?

Sisimai is the successor of bounceHammer, is a Mail Analyzing Interface: a Perl module or a Ruby Gem ( under the development ) for parsing RFC5322 bounce emails and for generating structured data such as JSON.

Parse Bounce Emails

Sisimai is easy to install and a library to parse bounce emails in UNIX mbox file or Maildir/ directory.

Getting Started

Detect Bounce Reasons

Sisimai can detect 25 bounce reasons such as "userunknown", "mailboxfull", "blocked", and so on.

vs. bounceHammer

On August 2014, We have started to develop a Perl version of Sisimai to resolve many fundamental problems of bounceHammer which are related to analytical precision ( it could not be detected 2 or more bounces in a single email ), many dependencies of external modules ( some modules have not maintained for a long time ), and cannot work on Perl 5.16 or later.
As of the present, the Perl version Sisimai is about 1.7 times faster than bounceHammer, have 2.0 times higher of analytical precision, depends only 2 external modules, and can work on Perl 5.10.1 or later versions of Perl. From August 2015, We have started to develop a Ruby version of Sisimai. The software will be released until the next spring.
